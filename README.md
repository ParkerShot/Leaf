# Notes — приложение для Windows

Заметки в стиле Apple: папки, поиск, закреплённые, чек-листы, таблицы,
картинки, форматирование, корзина, блокировка паролем, тёмная тема,
переключение языка (RU/EN). Все данные хранятся только на вашем компьютере.

## Что внутри
- main.js     — окно приложения, сохранение в файл, шифрование
- preload.js  — мост Electron
- index.html  — разметка (подключает styles.css и app.js)
- styles.css  — все стили
- app.js      — вся логика
- icon.ico    — иконка приложения
- fonts/      — шрифт Inter (зашит, работает без интернета)

ВАЖНО: index.html, styles.css и app.js работают только ВСЕ ВМЕСТЕ.

--------------------------------------------------

## Нужно (один раз)
Только Node.js (у вас уже установлен).

## Шаг 1. Распакуйте архив
Распакуйте в любую папку (например, на Рабочий стол).

## Шаг 2. Откройте командную строку В этой папке
Откройте папку -> кликните в адресной строке проводника ->
напечатайте  cmd  -> Enter.

## Шаг 3. Установите зависимости
    npm install
Скачается Electron (1-3 мин, нужен интернет). Жёлтые "warn" — это норма.

## Шаг 4. Запустите и проверьте
    npm start
Откроется приложение Notes.

## Шаг 5. Соберите установщик .exe
    npm run build
Появится папка dist с файлом "Notes Setup 1.0.0.exe".

--------------------------------------------------

## Если сборка падает на скачивании (таймаут GitHub)
Перед "npm run build" задайте зеркало (в том же окне):
    set ELECTRON_MIRROR=https://npmmirror.com/mirrors/electron/
    set ELECTRON_BUILDER_BINARIES_MIRROR=https://npmmirror.com/mirrors/electron-builder-binaries/

## Если ошибка "Cannot create symbolic link"
Включите Режим разработчика: Параметры (Win+I) ->
Конфиденциальность и защита -> Для разработчиков -> Режим разработчика.
Затем снова "npm run build".

## Где хранятся заметки
C:\Users\ВашеИмя\AppData\Roaming\Notes\notes-data.json
Заметки под паролем — в зашифрованном виде (AES-256).
Если забыть пароль, заметку не открыть (восстановления нет).
